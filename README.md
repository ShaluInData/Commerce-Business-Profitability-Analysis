# Commerce-Business-Profitability-Analysis

**About Dataset**
📊 Dataset Description
This dataset contains 3,500 records of e-commerce transactions, capturing sales performance across different products, categories, and regions. Each row represents an individual order and includes key business metrics such as quantity sold, total sales value, and profit generated. The dataset is designed to support analysis of product performance, regional trends, and overall business profitability.
It provides a structured view of how different product categories perform across various regions, enabling insights into revenue generation, cost efficiency, and customer purchasing behavior. The presence of both sales and profit metrics makes it particularly useful for evaluating margins and identifying high-performing or underperforming segments.

For a detailed exploratory analysis and visualization of this dataset, refer to the notebook:

**Columns Description**
Order Date: The date on which the transaction was recorded.

Category: The category to which the product belongs.
Region: The geographical region where the sale occurred.
Quantity: The number of units sold in the transaction.
Sales: The total revenue generated from the transaction.
Profit: The net profit earned from the transaction.

# E-Commerce Profit Analysis

## Project Overview

This project focuses on analyzing an e-commerce sales dataset to evaluate business performance, profitability, and regional trends. The analysis was performed using SQL for data cleaning and transformation, and the results can be visualized using Power BI dashboards.

The objective of this project is to generate actionable business insights such as identifying top-performing products, loss-making items, and regional sales performance.

---

## Dataset Description

The dataset contains transactional sales data from an e-commerce business. It includes information about products, categories, regions, sales amounts, quantities, and profit values.

Dataset Name: **ecom_sales**

---

## Table Structure

| Column Name  | Data Type | Description                                     |

| id           | INT       | Unique identifier for each record (Primary Key) |

| quantity     | INT       | Number of units sold                            |

## Key SQL Analysis Questions

The following business questions were answered using SQL queries:

1. What is the total sales generated by the business?
2. What is the total profit earned by the business?
3. Which products are generating negative profit?
4. Which region generates the highest total profit?
5. What percentage of total sales is contributed by each category?
6. What are the top 5 products based on total sales?
7. What is the total profit generated in each region?
8. What is the monthly sales trend?
9. Which category has the highest total profit?
10. What is the average sales value per order?
11. Which products have high sales but low profit?
12. What is the total quantity sold by category?
13. Which region has the lowest profit?
14. What is the profit margin for each category?
15. Identify loss-making products.
